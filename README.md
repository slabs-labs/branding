# Branding

Logos, colours and visual assets that represent slabs labs.

## Installation

You can install the branding assets via git:

```bash
git clone https://github.com/slab-labs/branding.git
```

## Contributing

Contributions to the branding assets are welcome! Please fork the repository and submit a pull request with your changes. Only minor fixes and additions are allowed without prior approval. For major changes, please open an issue first to discuss your ideas.
